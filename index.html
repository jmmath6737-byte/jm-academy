🔑 로그인 · 계정 관리
계정 추가/비밀번호 변경/퇴직 처리 방법
  1. 설정·관리 화면 아래쪽 "🔑 강사·직원 계정 관리" 카드로 이동
  2. 새 선생님이 오셨다면 이름·아이디·비밀번호를 입력하고 "+ 계정 추가" 클릭
  3. 비밀번호를 바꾸려면 해당 계정 옆 "비밀번호 변경" 클릭 후 새 비밀번호 입력
  4. 퇴직하신 선생님은 "삭제(퇴직)" 클릭 — 그 계정으로는 더 이상 로그인 불가
※ 본인이 로그인한 계정과, 마지막 남은 원장 계정은 삭제할 수 없도록 안전장치가 되어 있어요.
👨‍🎓 원생 등록 · 학원비
원생 등록 시 강사 선택 방법
  1. 원생관리 > "+ 원생 등록" 클릭
  2. "수강 과목" 입력창 아래에 등록된 과목·강사 버튼들이 나옵니다
  3. 해당하는 과목·강사를 클릭하면 파란색으로 바뀌며 자동 입력됨 (여러 개 선택 가능)
학원비는 원장 계정에서만 보이고 입력할 수 있어요. 선생님 계정으로 학생을 수정해도 기존 학원비 값은 그대로 유지됩니다.
💰 수납관리 사용법
  1. 사이드바 "수납관리" 진입 (원장 계정만 보임)
  2. 상단 월 선택기로 확인하고 싶은 달을 선택
  3. 학생 목록에서 "수납 체크" 클릭 → 금액·납부일·납부방법 입력 후 저장
  4. 잘못 눌렀다면 "미납처리"로 되돌리기 가능
  5. 하단에서 강사별 월별 회원수, 재원생/퇴원생 수, 퇴원율 자동 확인
📅 수업표 관리 (반복 시간표 vs 보강)
꼭 알아두세요: "수업표" 화면은 매주 반복되는 고정 시간표 하나뿐입니다.
  • 정규 수업 변경 (예: 요일·시간 자체가 바뀜) → 수업표 화면에서 직접 수정 = 매주 계속 적용
  • 하루짜리 보강 수업 → 수업표에 직접 추가하지 말고, 대시보드/학습관리의 "보강 등록" 기능 사용 → 그 날짜에만 기록되고 다음 주에 자동으로 사라지지 않음
수업표 엑셀(CSV) 다운로드·업로드 방법
  1. 설정·관리 > "📆 수업표 엑셀(CSV) 다운로드·업로드" 카드로 이동
  2. "📥 수업표 CSV 다운로드" 클릭 → 엑셀에서 파일 열기
  3. 엑셀에서 학생목록 열의 이름을 쉼표로 구분해 수정 (예: 김민수, 이서연)
  4. 저장 후, 같은 카드의 "📤 CSV 업로드해서 반영"으로 파일 선택
  5. 화면에 "반영 완료 N건 갱신" 메시지가 뜨면 완료
※ 새 과목·강사(열)는 CSV로 만들 수 없어요 — "과목·강사 관리"에서 먼저 추가한 뒤 CSV를 받아야 그 열이 나타납니다. 새 시간대(행)는 CSV에 적어 올리면 자동 추가됩니다.
🗂 학원업무 · 신규상담 등록율
  1. 학원업무 > "상담스케줄" 탭에서 구분을 "신규상담"으로 선택해 등록
  2. 등록된 항목 옆 "⬜ 미등록" 버튼을 눌러 실제 등록(입회)으로 이어지면 "✅ 등록완료"로 전환
  3. 탭 상단에 신규상담 등록율(%)이 자동으로 계산되어 표시됩니다
📥 학부모 온라인 상담·등록 신청
학부모용 신청 링크: 배포 주소 뒤에 ?apply=1을 붙여서 안내하세요 (예: https://jm-academy-xxxx.vercel.app/?apply=1). 로그인 없이 누구나 열람 가능합니다.
  1. 학부모가 신청서를 작성하고 개인정보 동의 후 제출
  2. 학원업무 > "상담스케줄" 탭에서 접수된 신청 확인 (새 신청이 있으면 사이드바 "학원업무" 메뉴에 빨간 배지 숫자로 표시되고, 화면을 켜두면 알림음도 울립니다)
  3. "원생으로 등록" 클릭 → 신청 내용이 자동으로 채워진 원생 등록창이 열림 → 확인 후 저장
  4. 처리 완료된 신청은 "삭제"로 목록에서 정리
카카오톡 채널 추가 버튼 설정: 신청 완료 화면에 "카카오톡 채널 추가" 버튼을 띄우려면, 코드 상단의 KAKAO_CHANNEL_URL 값에 학원 채널 URL을 입력해야 합니다 (카카오톡 채널 관리자센터 > 상세설정 > 채널 URL에서 복사). 비워두면 버튼이 보이지 않습니다.
